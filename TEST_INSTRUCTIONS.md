# Quick Debug Instructions

## Check Browser Console (F12)

1. Open browser: http://localhost:3000
2. Press F12 (or Cmd+Option+I)
3. Go to Console tab
4. Look for any RED errors
5. Take screenshot and share

## Check if Component Loads

In console, type:
```javascript
console.log("Test")
```

Should print "Test"

## Check React DevTools

1. Install React DevTools extension
2. Check component tree
3. Find EditProfilePage
4. Check if AddExperienceDialog exists
5. Check expDialogOpen state

## Quick Fix - Try This

1. Hard refresh: Cmd + Shift + R
2. Clear cache: Cmd + Shift + Delete
3. Close all localhost:3000 tabs
4. Reopen browser
5. Go to http://localhost:3000
6. Try again

If still not working, the frontend might need restart.
